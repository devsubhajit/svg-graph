About line graph

It is really very hard for a HTML coder like me to edit a line chart from a downloaded library. I had tried many but was not good enough for me. So I have created this line graph/chart. Its very easy to use for all HTML coder.

  • You can change the line color
  • You can also change the background and grid colors
  • you can set the fill colors
  • you can set the fill colors also with multiple gradients
  • you can curve the line (I never found a curved line graph)
  • you can take decision whether your graph will have the dots or not
  • responsive
  • Being an Animator I find animation everywhere and try to put it
  • no need of external libraries
  • Less Coding
